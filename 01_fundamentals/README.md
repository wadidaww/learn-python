# Module 01: Fundamentals

Build a solid Python foundation before advancing to more complex topics.

## Topics Covered

| File | Concepts |
|------|----------|
| `01_syntax_basics.py` | Variables, types, control flow, functions, comprehensions |
| `02_data_structures.py` | list, tuple, dict, set — operations and idioms |
| `03_oop.py` | Classes, inheritance, polymorphism, dataclasses, properties |
| `04_error_handling.py` | Custom exceptions, context managers, error patterns |
| `05_file_io.py` | File I/O, CSV, JSON, pathlib |

## Projects

- **`projects/todo_app.py`** — Full CLI todo app (argparse + JSON persistence)
- **`projects/log_analyzer.py`** — Log parser with error counting and reports

## Running

```bash
python 01_syntax_basics.py
python projects/todo_app.py --help
pytest tests/ -v
```
