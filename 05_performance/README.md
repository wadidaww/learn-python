# Module 05: Performance & Optimization

Learn to measure, profile, and speed up Python programs.

## Topics Covered

| File | Concepts |
|------|----------|
| `profiling_demo.py` | cProfile, timeit, memory profiling patterns |
| `async_demo.py` | asyncio: tasks, gather, queues, semaphores |
| `multiprocessing_demo.py` | multiprocessing Pool, Queue, shared memory |
| `projects/parallel_processor.py` | concurrent.futures for parallel I/O and CPU work |

## Running

```bash
python profiling_demo.py
python async_demo.py
python projects/parallel_processor.py
```
