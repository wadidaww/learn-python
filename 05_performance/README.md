# Module 05: Performance & Optimization

Learn to measure, profile, and speed up Python programs.

## Topics Covered

| File | Concepts |
|------|----------|
| `profiling_demo.py` | cProfile, timeit, memory profiling patterns |
| `async_demo.py` | asyncio: tasks, gather, queues, semaphores |
| `multiprocessing_demo.py` | multiprocessing Pool, Queue, shared memory |
| `projects/parallel_processor.py` | concurrent.futures for parallel I/O and CPU work |
| `projects/hpc_project/` | **HPC project** – vectorized ops, memory efficiency, caching, parallel matrix multiply |

## Running

```bash
python profiling_demo.py
python async_demo.py
python projects/parallel_processor.py

# HPC project (full demo suite)
python projects/hpc_project/main.py

# HPC project (individual modules)
python projects/hpc_project/benchmark.py
python projects/hpc_project/vectorized_ops.py
python projects/hpc_project/memory_efficient.py
python projects/hpc_project/caching.py
python projects/hpc_project/parallel_matrix.py
```
