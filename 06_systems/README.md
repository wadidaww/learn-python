# Module 06: Systems Programming

Build low-level network and OS tools using Python's standard library.

## Topics Covered

| File | Concepts |
|------|----------|
| `networking/tcp_server.py` | Socket programming, TCP server/client |
| `networking/http_server.py` | Minimal HTTP/1.1 server from scratch |
| `os_interaction/process_manager.py` | subprocess, process management |
| `os_interaction/task_scheduler.py` | Threading-based task scheduler |

## Running

```bash
python networking/tcp_server.py         # starts server on :9000
python os_interaction/task_scheduler.py # runs scheduled tasks
```
